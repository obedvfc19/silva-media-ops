# Silva Media Operations Suite

Automation tools built for Silva Media's lead management 
and bot monitoring workflows.

## Tools
- lead_alert_monitor.py — Detects inactive leads in GHL pipeline
- silence_detector.py — Monitors Assistable "Mia" uptime
- dashboard.html — Real-time operations overview
- silva_media_data.py — Synthetic data layer (production: GHL API)

## Stack
Python · GoHighLevel API · Zapier Webhooks · ClickUp · Claude
